# Antipodal Explorer

An interactive 3D globe that finds the antipodal (diametrically opposite) point of any location on Earth. Enter coordinates or detect your location, and see both points visualized on a real-time WebGL globe.

## Features

- Interactive 3D Earth globe with Three.js
- Enter coordinates manually or auto-detect via GPS/IP
- Antipodal point calculation with arc line visualization
- Reverse geocoding for place names
- Responsive design with mobile bottom sheet
- Smooth camera animations

## Tech Stack

- React 19 + TypeScript
- Three.js / React Three Fiber
- Tailwind CSS v4
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Type-check and build for production
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint
